# it-sphere-forum

# Портал-форум IT-сфера

Учебный проект по дисциплине.

## Описание
Разработан простой модуль для ввода и обработки информации о сообщениях форума.

## Используемые библиотеки
- FastAPI
- uvicorn[standard]
- SQLAlchemy

## Возможности модуля
- добавление сообщения
- просмотр всех сообщений
- поиск сообщений по автору
